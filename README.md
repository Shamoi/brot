# brot
Yet another bot for VK messages
# Installation
### Install vk_api and requests via pip.
`pip install vk_api requests`
### Create config file with:
`{
  "vk-login" : "79687426344",
  "vk-password" : "gbZTYznTk6iHDhvY81Xm",
  "chat-id" : 1,
  "ignored-users" : [1, 2]
}
`
### Run
`python3 run.py`
